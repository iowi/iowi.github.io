Debian系统的调试记录

发表于 2022-01-03 更新于 2022-01-05
1.固定IP地址

ip address #显示当前ip

nano /etc/network/interfaces

2.修改DNS
3.查看debian版本
4.安装speedtest
5.挂载分区

fdisk -l #查看硬盘
df -h #查看挂载情况
blkid #得到各盘的UUID=1b59220a-b571-6343-acc2-f89c173ab821

nano /etc/fstab #修改fstab

6.安装samba

apt install samba -y #安装samba服务

编辑Samba服务的设置

nano /etc/samba/smb.conf

编辑好之后设置增加samba用户和增加samba用户密码

创建samba用户和增加密码

useradd smb
smbpasswd -a smb

设置文件目录权限
重启samba服务

systemctl restart smbd
systemctl restart nmbd

Debian系统的调试记录

nano /etc/network/interfaces

3.查看debian版本
4.安装speedtest
5.挂载分区

fdisk -l #查看硬盘
df -h #查看挂载情况
ls -l /dev/disk/by-uuid #得到各盘的UUID=1b59220a-b571-6343-acc2-f89c173ab821

nano /etc/fstab #修改fstab

6.安装samba

apt install samba -y #安装samba服务

编辑Samba服务的设置

nano /etc/samba/smb.conf

编辑好之后设置增加samba用户和增加samba用户密码

创建samba用户和增加密码

useradd smb
smbpasswd -a smb

设置文件目录权限
重启samba服务

systemctl restart smbd
systemctl restart nmbd

8.安装AdGuardHome
9.定时重启容器
安装Crontab

apt install cron

编辑定时自动脚本

crontab -e

/etc/init.d/cron restart #重启任务
